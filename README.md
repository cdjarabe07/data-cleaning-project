# Data Cleaning Project

## Objective
The goal of this project is to clean and prepare a raw dataset for analysis by handling missing values, inconsistent formats, and data quality issues.

## Tools Used
- Python (Pandas, NumPy)

## Process
- Data loading and inspection
- Handling missing values
- Removing duplicates
- Standardizing column formats
- Converting data types (dates, numeric values)
- Cleaning categorical variables

## Key Improvements
- Cleaned inconsistent text data
- Standardized date formats
- Removed duplicates
- Created a new column (full_name) for better usability

## Result
A clean and structured dataset ready for analysis.

## How to Run
1. Open the notebook
2. Run all cells
3. Check cleaned dataset in `/data/raw_data.csv`
